# INSTALACIÓN RÁPIDA / QUICK START

## 🇪🇸 Instalación Rápida en Español

### En 5 pasos:

1. **Obtener Tailscale Auth Key**
   - Ir a: https://tailscale.com/admin/authkeys
   - Copiar un Auth Key activo

2. **Agregar Secret a GitHub**
   - Repository → Settings → Secrets and variables → Actions
   - Nuevo secret: `TAILSCALE_AUTH_KEY` = [pegar el key]

3. **Ejecutar Workflow**
   - Ir a: Actions → RDP → Run workflow → Confirmar

4. **Esperar Resultados**
   - Workflow tardará ~5-10 minutos
   - Ir a log de "Maintain Connection"

5. **Conectar por RDP**
   - Obtener: Address, Username (RDP), Password
   - Abrir: Conexión a Escritorio Remoto
   - Pegar dirección IP de Tailscale

**Resultado:** ¡Tu escritorio remoto estará listo! 🎉

## 🇬🇧 Quick Installation in English

### In 5 steps:

1. **Get Tailscale Auth Key**
   - Go to: https://tailscale.com/admin/authkeys
   - Copy an active Auth Key

2. **Add Secret to GitHub**
   - Repository → Settings → Secrets and variables → Actions
   - New secret: `TAILSCALE_AUTH_KEY` = [paste the key]

3. **Run Workflow**
   - Go to: Actions → RDP → Run workflow → Confirm

4. **Wait for Results**
   - Workflow will take ~5-10 minutes
   - Go to "Maintain Connection" log

5. **Connect via RDP**
   - Get: Address, Username (RDP), Password
   - Open: Remote Desktop Connection
   - Paste Tailscale IP address

**Result:** Your remote desktop will be ready! 🎉

## 🆘 Solución de Problemas / Troubleshooting

### 🇪🇸

**Problema: Workflow falla inmediatamente**
- ✓ Verifica que `TAILSCALE_AUTH_KEY` esté en Secrets
- ✓ Copia exactamente el Auth Key (sin espacios)
- ✓ El Auth Key debe estar activo en Tailscale

**Problema: No puedo conectarme por RDP**
- ✓ Verifica que copying la IP correcta
- ✓ Asegúrate de estar en la misma red Tailscale
- ✓ El usuario es `RDP` exactamente
- ✓ La contraseña es la generada por el workflow

**Problema: Workflow se cancela después de 60 minutos**
- ✓ Es normal (límite de GitHub Actions)
- ✓ Vuelve a ejecutar para reconectar

---

### 🇬🇧

**Problem: Workflow fails immediately**
- ✓ Check that `TAILSCALE_AUTH_KEY` is in Secrets
- ✓ Copy the Auth Key exactly (no spaces)
- ✓ The Auth Key must be active in Tailscale

**Problem: I can't connect via RDP**
- ✓ Verify you're copying the correct IP
- ✓ Make sure you're on the same Tailscale network
- ✓ Username is exactly `RDP`
- ✓ Password is the one generated by workflow

**Problem: Workflow cancels after 60 minutes**
- ✓ That's normal (GitHub Actions limit)
- ✓ Run it again to reconnect
